MXFramework

Yet another simple, lightweight, well-structured, Object-Oriented JavaScript framework.

Download Getting started »

Object Oriented

Writing JavaScript code, but still missing namespaces, classes, events, inheritance and polymorphism? MXFramework gives you a elegant but simple way to write and organize your code with object-oriented style, just like what you do with Java and C#.

Learn more »

JS Extensions

Getting troubles with formatting/parsing date, array manipulation and localization? MXFramework provides extensions to String, Date, Number and other build-in types, along with may useful utilities. What's more, you can use array as mutable set or even collection.

Learn more »

Controls and Apps

MXFramework is designed to be a fundamental framework other than a library. Instead of providing bunch of UI controls, it offers its own way to define and combine the UI elements, meanwhile it also supports Application-based development and running mode.
